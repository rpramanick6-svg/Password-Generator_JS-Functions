# Password-Generator_JS-Functions

I built this project as a simple Password Generator App using HTML, CSS, and JavaScript. The application allows users to generate random passwords based on selected criteria such as length, uppercase letters, lowercase letters, numbers, symbols, spaces, and duplicate exclusion.

## Features

- Generate random passwords based on selected options
- Adjust password length using a slider
- Include lowercase, uppercase, numbers, and symbols
- Option to exclude duplicate characters
- Option to include spaces
- Copy generated password to clipboard
- Password strength indicator

## My Contribution

I primarily focused on building the JavaScript functionality for this project. I worked on the password generation logic, option handling, clipboard copy feature, password strength indicator, and dynamic updates based on user input.

For the CSS styling and design, I took help from tutorials and AI assistance where needed. My main focus was on understanding and implementing the JavaScript logic behind the application.
